CryptoLook

  • Click to see code

Cryptocurrency Tracker App in React JS, Material UI and Chart JS with full responsive fuctionality. I used Context API to manage the state of the app and Coin Gecko API for backend.

What you can do on the app?

  • You can search coins (Name or shortcut)
  • You can click on the coin name and go on to the coin page which shows coins name, 24h/30d/etc Price Chart, rank of coin, market cap, information about coin, symbol and coin price
  • You can change the currency of prices(USD/EUR)

Which Informations app shows?

  • Coin name
  • Trending Coins
  • Price Chart
  • Coin symbol
  • Coin abbreviation
  • Coin price
  • Market Cap
  • Information about coin
  • 24 hours coin price change by percentage
